[中文](https://github.com/Akirako/drawio-desktop/edit/master/README.md) [English](https://github.com/Akirako/drawio-desktop/edit/master/README-en.md)

这是“Draw.io”的中文介绍，它是一个免费的绘图工具，您可以使用它创建流程图、布局图、网络图、图表和思维导图等等。

它可以在浏览器（例如Google Chrome）上操作，并且可以与Google Drive或Github合作保存创建的图形，。
